## compile:
  `python Linux_compile.py`   -   compiling

## run app:
  `python Linux_run.py`   -   run

## you can run the program without using python:

using clang:

  `clang -o bin/name_app main.cpp`  -  compiling
  
  `./bin/main`  -  run

or using make:

  `make main`   -   compiling
  
  `./main`   -   run

or:

`gcc -o main main.cpp`   -   compiling
  
  `./bin/main`   -   run
